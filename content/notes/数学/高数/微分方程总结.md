一份全景式的“常微分方程核心解法速查表”。在考场上，解题的第一步永远是**识别方程类型**，第二步才是**套用对应解法**。

------

### 一、 一阶常微分方程（核心：变量代换与分离）

这是所有复杂方程的求解终点。核心目标是将 $x$和 $y$彻底分开，或者凑成导数公式。

| **方程类型**                            | **标准形式特征**                                             | **核心解法套路 / 变量代换**                                  |
| --------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **可分离变量方程**                      | $g(y)dy = f(x)dx$或 $y' = f(x)g(y)$| **直接分离变量**：移项使两侧只有单一变量，然后直接两边积分。 |
| **齐次方程**                            | $y' = \varphi(\frac{y}{x})$或各项总次数相同                 | **令 $u = \frac{y}{x}$**，则 $y = ux$，$y' = u'x + u$。 代入后必然转化为关于 $u$和 $x$的可分离变量方程。 |
| **一阶线性非齐次**                      | $y' + P(x)y = Q(x)$($y$和 $y'$均为一次)                   | **通解公式法**：直接套用万能公式 $y = e^{-\int P(x)dx} \left[ \int Q(x)e^{\int P(x)dx} dx + C \right]$*(如果把 $x, y$互换后满足此形式，则公式里的 $x, y$对调)* |
| **伯努利方程**                          | $y' + P(x)y = Q(x)y^n$($n \neq 0, 1$)                       | **令 $z = y^{1-n}$**。 方程两边同除以 $y^n$，代入 $z$后，转化为关于 $z$的一阶线性微分方程，再套公式。 |
| **全微分方程** *(注：也可归为凑微分法)* | $P(x,y)dx + Q(x,y)dy = 0$满足 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$| **凑全微分** 或 **偏积分法**。 将左端凑成某个二元函数 $u(x,y)$的全微分 $du(x,y) = 0$，通解即为 $u(x,y) = C$。 |

------

### 二、 可降阶的高阶微分方程（核心：降级打击）

这类方程阶数 $\ge 2$，但必然缺失了某些关键变量（如缺 $y$或缺 $x$）。核心是“缺谁补谁的导数”。

| **方程类型**             | **缺失变量**                                | **降阶代换法则**                               | **后续步骤**                                                 |
| ------------------------ | ------------------------------------------- | ---------------------------------------------- | ------------------------------------------------------------ |
| **纯 $x$型**            | 缺 $y$及其低阶导数 $y^{(n)} = f(x)$| 无需代换                                       | **连续积分法**：对方程右端连续积 $n$次，每次产生一个新常数。 |
| **缺 $y$型**            | 缺 $y$(显含 $x, y', y''$) $y'' = f(x, y')$| **令 $y' = p$** 则 $y'' = p'$| 化为一阶方程 $p' = f(x, p)$。解出 $p$后，再积分一次 $y = \int p dx$。 |
| **缺 $x$型** *(最易错)* | 缺 $x$(显含 $y, y', y''$) $y'' = f(y, y')$| **令 $y' = p$** 则 **$y'' = p \frac{dp}{dy}$** | 化为关于 $p$和 $y$的一阶方程 $p \frac{dp}{dy} = f(y, p)$。解出 $p(y)$后，分离变量解 $\frac{dy}{p(y)} = dx$。 |

------

### 三、 高阶常系数线性微分方程（核心：代数映射）

这是考研综合大题的绝对主力。口诀是：**“非齐次通解 = 齐次通解 $Y$+ 非齐次特解 $y^\*$”**。

#### 1. 求齐次通解 $Y$（写特征方程）

对于方程 $y'' + py' + qy = 0$，写出特征方程 $r^2 + pr + q = 0$：

- **两个不相等实根 $r_1 \neq r_2$**：$Y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$
- **二重实根 $r_1 = r_2 = r$**：$Y = (C_1 + C_2 x) e^{r x}$
- **一对共轭复根 $r = \alpha \pm \beta i$**：$Y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)$

#### 2. 求非齐次特解 $y^*$（待定系数法）

对于方程 $y'' + py' + qy = f(x)$，根据右端 $f(x)$的长相设特解，并代入原方程反求系数。

| **自由项 f(x) 形式**                                         | **提取的比对参数**          | **特解 y∗ 的设法与 k 的判定**                                |
| ------------------------------------------------------------ | --------------------------- | ------------------------------------------------------------ |
| **多项式 $\times$指数型** $P_m(x) e^{\lambda x}$| 提取 $\lambda$| $y^* = x^k \cdot Q_m(x) \cdot e^{\lambda x}$* $Q_m(x)$是设出的一般 $m$次多项式。 * **$k$值**：若 $\lambda$不是特征根取 0；是单根取 1；是重根取 2。 |
| **指数 $\times$三角震荡型** $e^{\lambda x} [P_l(x) \cos \omega x$ $+ P_n(x) \sin \omega x]$| 提取 $\lambda \pm \omega i$| $y^* = x^k \cdot e^{\lambda x} [R_m^{(1)}(x) \cos \omega x + R_m^{(2)}(x) \sin \omega x]$* $m = \max(l, n)$，必须同时设出 $\cos$和 $\sin$项。 * **$k$值**：若 $\lambda \pm \omega i$不是特征根取 0；是特征根取 1。 |

#### 3. 数学一专属考点：欧拉方程 (Euler Equation)

- **形式**：$x^n y^{(n)} + p_1 x^{n-1} y^{(n-1)} + \dots + p_{n-1} x y' + p_n y = f(x)$

- **解法**：作自变量代换 **$x = e^t$** (若 $x>0$)。记 $D = \frac{d}{dt}$，则：

  - $x y' = Dy$

  - $x^2 y'' = D(D-1)y$

  - $x^3 y''' = D(D-1)(D-2)y$

    代入后，它必然转化为一个关于自变量 $t$的**常系数线性微分方程**。解完后切记要把 $t = \ln x$ 代回。
