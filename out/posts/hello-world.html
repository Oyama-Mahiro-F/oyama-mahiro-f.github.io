# 博客

技术栈

组件 选型
框架 Astro v7
模板 AstroPaper v7
样式 Tailwind CSS v4
字体 Roboto Variable（本地） + 系统中文回退
搜索 Pagefind
代码高亮 Shiki（自定义 Typora GitHub 主题）
数学公式 KaTeX
部署 GitHub Actions → GitHub Pages

本地开发

# 安装依赖
cd astro-paper
pnpm install

# 启动开发服务器
pnpm dev
# → http://localhost:4321

# 构建
pnpm build

发布

# 推送即可，GitHub Actions 自动构建部署
./push.bat "提交信息"

目录结构

astro-paper/
├── src/
│   ├── content/posts/       # 文章 (.md)
│   ├── content/pages/       # 固定页面 (about.md)
│   ├── pages/               # 路由页面
│   ├── components/          # 组件
│   ├── styles/              # 样式 (theme.css, typography.css)
│   └── i18n/lang/           # 翻译 (zh-CN.ts)
├── public/fonts/            # 本地字体
└── astro-paper.config.ts    # 站点配置
